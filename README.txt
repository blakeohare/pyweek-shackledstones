Setup and dependencies
[GAMENAME] requires python 2.6 or python 3.1.  It also requires pygame.

Gameplay
Arrow keys to control your character, use A to advance text, make selctions on menus and use items bound to the A key.  
Press enter to bring up the in game inventory menu.  Assign items to a, s, d, and space.
w:enter wide screen
f: full screen
esc:quit
